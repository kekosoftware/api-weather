## About API

With this api you can check the weather of a city.


## How to

* git clone https://github.com/kekosoftware/api-weather.git
* composer install
* configure .env file
* Fill in the variable WEATHERSTACK_KEY with your key into the .env file
* php artisan key:generate
* php artisan migrate
* php artisan serve
* http://127.0.0.1:8000
